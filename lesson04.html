matplotlib（マットプロットリブ）とは？

matplotlibは、Pythonでグラフや図を描画するための定番ライブラリです。

「データの傾向を視覚的に見たい」「分析結果を図で表現したい」といったときに活用されます。

matplotlibを使うメリット

  • 折れ線グラフ・棒グラフ・円グラフなど、豊富な種類のグラフを描ける
  • 軸のラベルやタイトル、凡例など細かな調整が可能
  • 分析結果を視覚化して、直感的に理解できる
  • 他のライブラリ（pandasやNumPy）との連携も簡単

  • 折れ線グラフの作成

(1) import matplotlib.pyplot as plt
(2) import pandas as pd
(3)
(4) # データ作成
(5) df = pd.DataFrame({
(6)     '名前': ['太郎', '花子', '次郎', '三郎'],
(7)     '年齢': [23, 25, 22, 24],
(8)     '得点': [88, 92, 85, 90]
(9) })
(10)
(11) # 折れ線グラフ描画
(12) plt.figure(figsize=(8, 5))
(13) plt.plot(df['名前'], df['得点'], marker='o', linestyle='-', color='blue')
(14)
(15) # タイトルと軸ラベル
(16) plt.title('生徒の得点推移')
(17) plt.xlabel('名前')
(18) plt.ylabel('得点')
(19)
(20) # グリッド表示
(21) plt.grid(True)
(22)
(23) # グラフの表示
(24) plt.show()
実行結果：


  


・X軸：名前（太郎, 花子, 次郎, 三郎）
・Y軸：得点（85〜92の範囲）
・各点にマーカー（●）が付き、青色の線でつながれている
・タイトル：生徒の得点推移
・グリッド線あり

  • 散布図の作成

(1) import matplotlib.pyplot as plt
(2) import pandas as pd
(3)
(4) # データ作成（10人分）
(5) df = pd.DataFrame({
(6)     '名前': ['太郎', '花子', '次郎', '三郎', '美咲', '健一', '由紀', '翔太', '陽菜', '直人'],
(7)     '国語': [85, 90, 78, 92, 88, 76, 95, 84, 91, 87],
(8)     '数学': [80, 95, 85, 89, 86, 78, 96, 82, 90, 85]
(9) })
(10)
(11) # 散布図描画
(12) plt.figure(figsize=(10, 6))
(13) plt.scatter(df['国語'], df['数学'], color='blue', marker='o')
(14)
(15) plt.title('国語と数学の得点の散布図')
(16) plt.xlabel('国語の得点')
(17) plt.ylabel('数学の得点')
(18) plt.grid(True)
(19) plt.tight_layout()
(20) plt.show()
実行結果：




  


        

・X軸：国語の得点（約70～100）
・Y軸：数学の得点（約70～100）
・青丸の点で各生徒をプロット
・各点の近くに名前ラベル（太郎、花子、次郎…）
・タイトル：国語と数学の得点の散布図
・グリッド線あり

  • 折れ線グラフの線の種類と色について

(1) import matplotlib.pyplot as plt
(2) import pandas as pd
(3)
(4) # データ作成
(5) df = pd.DataFrame({
(6)     '名前': ['太郎', '花子', '次郎', '三郎'],
(7)     '得点': [88, 92, 85, 90]
(8) })
(9)
(10) plt.figure(figsize=(8, 5))
(11)
(12) # 折れ線グラフ（青・実線）
(13) plt.plot(df['名前'], df['得点'], color='blue', linestyle='-',
(14)          marker='o', label='実線（青）')
(15)
(16) # 折れ線グラフ（赤・破線）
(17) plt.plot(df['名前'], df['得点'] + 2, color='red', linestyle='--',
(18)          marker='s', label='破線（赤）')
(19)
(20) # 折れ線グラフ（緑・点線）
(21) plt.plot(df['名前'], df['得点'] + 4, color='green', linestyle=':',
(22)          marker='^', label='点線（緑）')
(23)
(24) plt.title('折れ線グラフの線の種類と色の例')
(25) plt.xlabel('名前')
(26) plt.ylabel('得点')
(27) plt.grid(True)
(28) plt.legend()
(29) plt.tight_layout()
(30) plt.show()
実行結果：


  


      
・3種類の折れ線グラフを重ねて表示
・青：実線（-）、丸マーカー（o）
・赤：破線（--）、四角マーカー（s）
・緑：点線（:）、三角マーカー（^）
・凡例付き、グリッド線あり
