seaborn（シーボーン）とは？

seabornは、Pythonで洗練されたグラフを簡単に描画できるライブラリです。

matplotlibをベースにしており、統計的なデータの可視化に特化しています。

seabornを使うメリット

  • 見た目が整った美しいグラフを簡単に描ける
  • カテゴリごとの比較・分布・相関などを視覚化しやすい
  • matplotlibよりもコードが簡潔で直感的
  • pandasのDataFrameとの相性が抜群
  • 色やスタイルがデフォルトで整っているので調整不要なことが多い

  • 折れ線グラフの作成

(1) import seaborn as sns
(2) import matplotlib.pyplot as plt
(3) import pandas as pd
(4)
(5) # データ作成
(6) df = pd.DataFrame({
(7)     'date': pd.date_range('2025-01-01', periods=7),
(8)     'sales': [120, 130, 125, 145, 150, 160, 170]
(9) })
(10)
(11) # スタイル設定
(12) sns.set(style="whitegrid")
(13)
(14) # 折れ線グラフの描画
(15) plt.figure(figsize=(9, 5))
(16) sns.lineplot(x='date', y='sales', data=df,
(17)              marker='o', color='blue')
(18)
(19) plt.title('日別売上の推移（Seaborn）')
(20) plt.xlabel('日付')
(21) plt.ylabel('売上（万円）')
(22) plt.show()
