seaborn（シーボーン）とは？

seabornは、Pythonで洗練されたグラフを簡単に描画できるライブラリです。

matplotlibをベースにしており、統計的なデータの可視化に特化しています。

seabornを使うメリット

  • 見た目が整った美しいグラフを簡単に描ける
  • カテゴリごとの比較・分布・相関などを視覚化しやすい
  • matplotlibよりもコードが簡潔で直感的
  • pandasのDataFrameとの相性が抜群
  • 色やスタイルがデフォルトで整っているので調整不要なことが多い

  • 折れ線グラフの作成

(1) import seaborn as sns
(2) import matplotlib.pyplot as plt
(3) import pandas as pd
(4)
(5) # データ作成
(6) df = pd.DataFrame({
(7)     'date': pd.date_range('2025-01-01', periods=7),
(8)     'sales': [120, 130, 125, 145, 150, 160, 170]
(9) })
(10)
(11) # スタイル設定
(12) sns.set(style="whitegrid")
(13)
(14) # 折れ線グラフの描画
(15) plt.figure(figsize=(9, 5))
(16) sns.lineplot(x='date', y='sales', data=df,
(17)              marker='o', color='blue')
(18)
(19) plt.title('日別売上の推移（Seaborn）')
(20) plt.xlabel('日付')
(21) plt.ylabel('売上（万円）')
(22) plt.show()
実行結果：


  


        
・日別売上をSeabornで折れ線グラフとして表示
・マーカー付き（○）、青い実線
・x軸は日付、y軸は売上
・白背景グリッド

  • 散布図の作成

(1) import seaborn as sns
(2) import matplotlib.pyplot as plt
(3) import pandas as pd
(4) import numpy as np
(5) # データ生成（身長と体重の関係）
(6) 身長 = np.array([168.2, 172.1, 159.2, 177.4, 153.1, 171.2, 173.3, 149.4, 148.7, 158.3])
(7) 身長_m = 身長 / 100  # 身長(m)
(8) BMI = np.array([19.82, 22.24, 20.55, 22.23, 21.45, 21.36, 24.64, 20.04, 23.08, 23.87])
(9) 体重 = np.round(BMI * 身長_m**2, 1)  # 体重（kg）
(10) # DataFrameにまとめる
(11) df = pd.DataFrame({
(12)     'height': 身長,
(13)     'BMI': BMI,
(14)     'weight': 体重
(15) })
(16) # スタイル設定
(17) sns.set(style="whitegrid")
(18) # 散布図の描画
(19) plt.figure(figsize=(8, 5))
(20) sns.scatterplot(x='height', y='weight', data=df, marker='o')(22) plt.xlabel('height')
(21) plt.ylabel('weight')
(22) plt.show()
実行結果：


  

       
・身長と体重の関係を表す散布図を描画
・マーカーは丸
・白いグリッド背景（whitegridスタイル）

  • 箱ひげ図の作成

(1) import seaborn as sns
(2) import matplotlib.pyplot as plt
(3) import pandas as pd
(5) # データ作成（科目別の点数）
(6) df = pd.DataFrame({

(10) })
(11) # データを縦長に変換（seaborn用）

(13) # 箱ひげ図の描画
(14) plt.figure(figsize=(8, 5))
(15) sns.boxplot(x='subject', y='score', data=df_melted)

(17) plt.xlabel('subject')
(18) plt.ylabel('score')
(19) plt.tight_layout()
(20) plt.show()
実行結果：

        

  

       
・国語・数学・英語の点数分布を箱ひげ図で可視化
・箱の範囲は四分位範囲（25〜75%）、中央線は中央値
・外れ値（離れている点）がある場合も表示

  • ヒートマップの作成

(1) import seaborn as sns
(2) import matplotlib.pyplot as plt
(3) import pandas as pd
(4) # サンプルデータ作成（各教科の点数）
(5) df = pd.DataFrame({
(6)     'Japanese': [70, 82, 91, 65, 77, 83, 69, 85, 90, 75],
(7)     'Mathematic': [68, 62, 58, 70, 65, 72, 66, 68, 75, 64],
(8)     'English': [85, 80, 89, 78, 82, 87, 90, 83, 79, 88],
(9)    'Science': [72, 75, 70, 68, 77, 76, 73, 78, 80, 74],
(10)    'Social': [88, 85, 83, 90, 89, 92, 87, 91, 86, 84]
(11) })
(12) # 相関係数の算出
(13) corr = df.corr()
(14) # ヒートマップの描画
(15) plt.figure(figsize=(8, 6))
(16) sns.heatmap(corr, annot=True, cmap='coolwarm', square=True, fmt=".2f", linewidths=0.5)
(17) plt.title('教科間の相関係数ヒートマップ')
(18) plt.tight_layout()
(19) plt.show()
実行結果：


  

       
・各教科の相関関係をヒートマップで可視化  
・値は相関係数（-1〜+1）、色は coolwarm カラーマップ  
・annot=True で数値をマス内に表示  
・日本語ラベルも可視化（japanize_matplotlib）
